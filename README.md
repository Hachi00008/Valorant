**ウルトポイントとは？**\
キル、マップ中に落ちている丸い球体、スパイクの設置／解除で得られるポイント。\
キャラごとに設定されているポイント数を超えると「ウルト」が使えるようになる。\
なお、このポイントは試合中ずっと保つことができ、発動ができる際に余分に貯めることはできない。

**🤞 ←　このマークは？**\
エージェントによっては、購入する必要はないが一度使用すると2キルするまで使用が禁止されるアビリティがあります。\
このマークは、そのアビリティを指しています。\
※ソーヴァは、1分に1回使用できる仕様になっています※\
※セージは、45分に1回使用できる使用になっています※
